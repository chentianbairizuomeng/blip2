# Mini-BLIP2 Reproduction Package

File structure:

```text
blip2/
├── README.md
├── index.html
├── requirements.md
├── code/
└── report/
    └── report_template.md
```

Description:

- `index.html`: Web version of the reproduction requirements. Open it directly in a browser.
- `requirements.md`: Markdown version of the reproduction requirements.
- `code/`: Put implementation code here.
- `report/report_template.md`: Experiment report template.

Students should start by reading `index.html` or `requirements.md`.
